Design Inheritance Points from Two PPTs (English Summary)
Document 1: 2025 Microsoft Environmental Sustainability Report
1. Navigation & Structure
Fixed multi‑level navigation bar: Top horizontal menu with core sections (Overview, Carbon, Water, Waste, Ecosystems, Microsoft sustainability, Customer sustainability, Global sustainability, Appendix) for clear chapter switching.
Page footer consistent labels: Repeats Overview | Microsoft sustainability | Customer sustainability | Global sustainability | Appendix + core pillars (Carbon/Water/Waste/Ecosystems) on every page for orientation.
Numbered page hierarchy: Clear page numbers + section titles to show report structure and reading progress.
2. Highlighting Style
Bold key phrases & data: Highlights core achievements, percentages, and metrics (e.g., 90.9% reuse rate, 85.3% waste diversion).
Table highlighting: Uses clean tables to display progress data, targets, and comparisons with labeled titles (e.g., Carbon Table 1, Waste Table 2).
Block captions: Uses short, strong labels like INNOVATION IN ACTION, PARTNERING FOR IMPACT, LEARNINGS AND WHAT’S NEXT to mark thematic modules.
3. White Space & Layout
Generous margins & breathing room: Ample white space around text, charts, and images to avoid clutter.
Modular grid layout: Content split into compact, scannable blocks (text + charts + tables) with consistent alignment.
Visual balance: Equal distribution of text, data visuals, and headings to maintain readability.
4. Font Hierarchy
Title font: Large, bold, sans-serif for main report title and section headers.
Section headings: Medium bold, distinct size to separate chapters.
Body text: Clean, readable sans-serif, standard size for paragraphs; bullet points for concise lists.
Data labels: Small but clear font for charts, tables, and annotations.
5. Other Design Features
Consistent color theme: Unified color palette matching Microsoft’s sustainability branding (green/blue tones).
Icon & visual consistency: Simple, flat icons for Carbon/Water/Waste/Ecosystems; professional data charts (bar/line/pie).
Bullet-point readability: Short bullet points instead of long paragraphs for quick scanning.
Document 2: Singapore’s Urban Development PPT
1. Navigation & Structure
Vertical side navigation bar: Fixed left sidebar with section labels (BACKGROUND, SPATIAL DIAGRAM, LAND USE, REVITALIZATION, REGENERATION, DISCUSSION) for persistent section tracking.
Section title repetition: Displays current section label on each slide to show position in the presentation flow.
2. Highlighting Style
Border & shading highlights: Uses colored boxes or underlines to emphasize key data, maps, and conclusions.
Image caption highlighting: Clear labels below maps/diagrams to mark sources and themes.
Bullet-point emphasis: Bold keywords within bullet points to draw attention to core ideas.
3. White Space & Layout
Map‑focused clean layout: Large central maps/diagrams with surrounding white space for clarity.
Balanced image‑text ratio: Images/maps take 50–70% of space; text kept concise to avoid crowding.
Grid alignment: Neat alignment of text boxes, charts, and maps for a tidy academic presentation style.
4. Font Hierarchy
Main title: Large, bold, uppercase or semi‑uppercase for topic emphasis.
Section headers: Medium bold, all caps for sidebar and slide titles.
Body text: Simple sans-serif, moderate size; short paragraphs and bullet points.
Data labels & captions: Small, neat font for charts, tables, and map sources.
5. Other Design Features
Academic‑style visuals: Clear maps, comparison charts, and timeline diagrams for planning content.
Before‑after comparison layout: Side‑by‑side images to show regeneration results.
Simple table design: Clean, bordered tables for land use and statistical data.
